## Key Project: Impact of FinTech on Traditional Banking in Nepal

- Primary survey (110+ responses, 2024–2025).
- Findings: Convenience/speed dominate adoption; ~90% reduced branch visits; satisfaction ~4/5; challenges include security/connectivity.
- Location analysis: Efficiency focus consistent, with regional variations (e.g., cost sensitivity in Pokhara).
